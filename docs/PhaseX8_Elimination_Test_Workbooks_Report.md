## What Was Created

10 elimination test workbooks were generated by manipulating the OOXML structure of the legacy ConMas Output Excel workbook (`FormTest - Copy.xlsx`). Each test modifies **exactly one variable** to isolate what the real ConMas runtime depends on.

### Workbook Structure (Legacy Baseline)

| Sheet | Visible? | Purpose |
|-------|----------|---------|
| `_Fields` | Hidden | Field metadata (headers only) |
| `Sheet1` | Visible | Content sheet: 5 merged regions, 33 cells, 6 comments |
| `ExcelOutputSetting` | Visible | ConMas XML configuration (36 rows) |

Defined names: `_xlnm.Print_Area = Sheet1!$A$1:$D$12`
